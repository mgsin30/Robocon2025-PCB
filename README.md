# Robocon2025-PCB
高専ロボコン2025のために開発した基板 <br>
各基板のファームウェアは [こちら](https://github.com/mgsin30/Robocon2025-Firmware)

## CAN Bus Hub
各基板を CAN バスで接続するためのハブ基板

## Converter Board
サーボモーターや LED テープなどのために電源電圧を降圧する基板

## GPIO Board
リミットスイッチやジャイロセンサー、その他 GPIO を使うための基板

## Motor Driver
ブラシ付きモーター用のモータードライバー

## Power Board
電源電圧の監視や非常停止の管理などを行う電源基板

## PWM Board
サーボモーターなどを使うための基板

## Remote Emergency Stop Board (TX)
遠隔非常停止の送信側基板

## Remote Emergency Stop Board (RX)
遠隔非常停止の受信側基板

## RoboMaster Board
ロボマスモーターを使うための基板

## Solenoid Driver
ソレノイドや電磁弁などを使うための基板

## USB-CAN Board
USB と CAN を変換する基板

## XT30 Hub
XT30 コネクタのハブ基板

## XT60 Hub
XT60 コネクタのハブ基板
